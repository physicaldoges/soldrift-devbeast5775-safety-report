Checklist

Before hiring any Solana or Web3 developer

Server Access

Use a limited server account, not unrestricted administrator access. Remove unknown access keys. Rotate passwords after handover. Check open ports, background services, scheduled jobs, process managers, and shell history.

Wallets and Keys

Never share seed phrases or long-term private keys. Treat all keys used during development as disposable. Use fresh wallets after any suspicious access.

Code Audit

Search for outbound requests, webhooks, messaging bots, dynamic code execution, hidden encoded data, encrypted blobs, wallet export routines, and unexpected storage paths.

Service Keys

Rotate infrastructure, repository, dashboard, encryption, database, and deployment keys after handover. Keep production secrets outside public repositories.

Payment Protection

Pay by milestone. Require source-code handover, installation instructions, dependency list, and a clear refund or remediation policy before large payments.

Evidence Hygiene

Keep transaction hashes, screenshots, message exports, repository commit history, and server command outputs. Redact secrets before publishing any report.
